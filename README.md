# Customer Segmentation using K-Means

## Project Overview

This project applies the K-Means clustering algorithm to segment customers based on their annual income and spending score.

## Dataset

Mall Customers Dataset

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook
